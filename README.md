# My API

## Deskripsi

API sederhana untuk manajemen pengguna yang di-deploy ke Vercel.

## Endpoint

### GET /api/users

Mengembalikan daftar pengguna statis.

### POST /api/users

Menambahkan pengguna baru ke dalam array sementara.
